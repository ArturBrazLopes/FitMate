# Como Rodar

## Método Simples
Execute: `rodar.bat`

## Manual
1. `config-mongodb.bat` - Configura MongoDB
2. `setup.bat` - Instala dependências
3. `start.bat` - Inicia servidores

## Outros Scripts
- `stop.bat` - Para servidores
- `start.bat` - Inicia tudo

## Acesso
Frontend: http://localhost:5173
Backend: http://localhost:4000
